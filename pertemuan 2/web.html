berkenalan dengn pemerograman web

web programming atau pemerograman web merupakan istilah yang erat kaitannya dengan internet dan website. memang benar ,karna pemerograman web merupakan suatu proses pembuatan website untuk keprluan internet. orang banyak mengenal web dengan isitlah www atau world wide web


world wide web adalah halaman - halaman websiste yang terkoneksi dengan lainnya atau di sebut jugan dengan hyperlink yang membentuk samudra informasi yang berjalan dengan protokolhyper transfer protocol (http).

kelebihan aliasi berbasis web :

  1. mudah di akses dan dapat di akses diberbagai jenis perangkat seperti prangkatmobile/smartphone,prangkat deskop dan prangkat table
  2. dapat di jalankan di berbagai sistem informsi (os) seperti mac ,windows ,linux,dll
  3. dapat di akses darimanasaja dan kapan saja ,asalkan terhubung dengan jaringan internet maupun jaringan lokal tempat aplikasi web itu berada

kekurangan aplikasi berbasis web

  • memerlukan koneksi internet yang stabil agar ttp dapat menggunakn aplikasi nya secara nyaman
  • memerlukan sistem keaman yang baik dari sisi server,website,koneksi yang digunakan,dll
